# Deskripsi Wireframe Teks

## 1. Beranda (Homepage)

### Layout Section-by-Section (Atas ke Bawah):

**A. Header (Sticky)**
- Logo sekolah di pojok kiri atas
- Navigation menu di tengah (Beranda, Tentang Kami, Akademik, PPDB, Berita, Kontak)
- Tombol "Login Siswa/Guru" di pojok kanan
- Mobile menu (hamburger) untuk layar kecil

**B. Hero Section**
- Banner besar dengan gambar/foto kegiatan sekolah
- Overlay teks: "Selamat Datang di [Nama Sekolah]"
- Subjudul singkat tentang visi sekolah
- Primary CTA: Tombol besar "Jelajahi Sekolah" (tengah)
- Secondary CTA: "Lihat PPDB" (di bawah primary CTA)

**C. Quick Info/Announcement Bar**
- Bar informasi penting dengan ikon (misalnya: "Pendaftaran dibuka sampai 30 Juni")
- Tautan ke halaman terkait

**D. Tentang Sekolah (Preview)**
- Judul: "Mengapa Memilih Kami?"
- Deskripsi singkat 2-3 kalimat
- Icon features (3-4 poin unggulan dengan ikon)
- CTA: Tombol "Selengkapnya" atau "Tentang Kami"

**E. Program Unggulan**
- Judul: "Program Unggulan Kami"
- 3 card program dengan gambar, judul, dan deskripsi singkat
- CTA: Tombol "Lihat Semua Program"

**F. Berita Terkini (Preview)**
- Judul: "Berita & Kegiatan Terbaru"
- 3 card berita terbaru dengan thumbnail, judul, tanggal, dan excerpt
- CTA: Tombol "Lihat Semua Berita"

**G. Testimoni**
- Judul: "Apa Kata Mereka?"
- Carousel testimoni dari siswa, orang tua, dan guru
- Nama, foto, dan jabatan

**H. Galeri Foto (Preview)**
- Judul: "Galeri Kegiatan"
- Grid 2x3 foto kegiatan sekolah
- CTA: Tombol "Lihat Galeri Lengkap"

**I. Footer**
- Alamat, kontak, dan jam operasional
- Social media links
- Quick links
- Newsletter signup (input email + tombol subscribe)

### Urutan Konten:
1. Header
2. Hero Section
3. Quick Info Bar
4. Tentang Sekolah
5. Program Unggulan
6. Berita Terkini
7. Testimoni
8. Galeri
9. Footer

### Posisi CTA:
1. Hero Section: "Jelajahi Sekolah" (primary), "Lihat PPDB" (secondary)
2. Tentang Sekolah: "Selengkapnya"
3. Program Unggulan: "Lihat Semua Program"
4. Berita: "Lihat Semua Berita"
5. Galeri: "Lihat Galeri Lengkap"
6. Footer: "Login Siswa/Guru"

## 2. Halaman PPDB

### Layout Section-by-Section (Atas ke Bawah):

**A. Header (Sticky)**
- Sama dengan header beranda

**B. Hero Section**
- Banner dengan tema pendaftaran
- Judul: "Penerimaan Peserta Didik Baru"
- Subjudul: Tahun ajaran dan periode pendaftaran
- Primary CTA: Tombol besar "Daftar Sekarang"

**C. Informasi Umum PPDB**
- Judul: "Informasi Umum"
- Timeline pendaftaran (dengan visual timeline)
- Syarat pendaftaran (list dengan checklist)
- Biaya pendaftaran (jika ada)

**D. Alur Pendaftaran**
- Judul: "Alur Pendaftaran"
- Step-by-step process dengan icon (4-5 langkah)
- Penjelasan singkat setiap langkah

**E. Jenis Pendaftaran**
- Judul: "Jalur Pendaftaran"
- Card untuk setiap jalur (Reguler, Beasiswa, Prestasi, dll.)
- Syarat khusus setiap jalur

**F. Formulir Pendaftaran (Preview)**
- Judul: "Mulai Pendaftaran"
- Preview form dengan field-field utama
- CTA: Tombol besar "Isi Formulir Pendaftaran"

**G. Dokumen yang Diperlukan**
- Judul: "Dokumen yang Harus Disiapkan"
- Checklist dokumen dengan icon

**H. Pertanyaan Umum (FAQ)**
- Judul: "Pertanyaan yang Sering Diajukan"
- Accordion dengan pertanyaan dan jawaban

**I. Kontak Bantuan**
- Judul: "Butuh Bantuan?"
- Informasi kontak PPDB (email, WA, nomor telepon)
- Jam operasional

**J. Footer**
- Sama dengan footer beranda

### Urutan Konten:
1. Header
2. Hero Section
3. Informasi Umum
4. Alur Pendaftaran
5. Jenis Pendaftaran
6. Formulir Pendaftaran
7. Dokumen yang Diperlukan
8. FAQ
9. Kontak Bantuan
10. Footer

### Posisi CTA:
1. Hero Section: "Daftar Sekarang" (primary)
2. Formulir Pendaftaran: "Isi Formulir Pendaftaran"
3. Setiap card jalur pendaftaran: "Daftar Jalur Ini"
4. Kontak Bantuan: Tombol "Hubungi Kami via WA"

## 3. Halaman Berita

### Layout Section-by-Section (Atas ke Bawah):

**A. Header (Sticky)**
- Sama dengan header beranda

**B. Hero Section**
- Judul: "Berita & Kegiatan"
- Breadcrumb navigation

**C. Filter & Search**
- Search bar untuk mencari berita
- Kategori filter (dropdown atau tabs)
- Sort options (terbaru, terpopuler)

**D. Featured News**
- Artikel berita utama dengan banner besar
- Judul, tanggal, kategori, dan excerpt
- CTA: Tombol "Baca Selengkapnya"

**E. Daftar Berita**
- Grid atau list berita (dengan pagination)
- Setiap card berita berisi:
  - Thumbnail gambar
  - Judul berita
  - Tanggal publikasi
  - Kategori
  - Excerpt singkat
  - CTA: Tombol "Baca"

**F. Sidebar (Desktop) / Accordion (Mobile)**
- Kategori berita
- Arsip berita (berdasarkan bulan/tahun)
- Berita populer
- Newsletter signup

**G. Pagination**
- Navigasi halaman berita

**H. Footer**
- Sama dengan footer beranda

### Urutan Konten:
1. Header
2. Hero Section
3. Filter & Search
4. Featured News
5. Daftar Berita
6. Sidebar/Widgets
7. Pagination
8. Footer

### Posisi CTA:
1. Featured News: "Baca Selengkapnya"
2. Setiap card berita dalam daftar: "Baca"
3. Sidebar: "Subscribe Newsletter"
4. Berita populer: "Baca" (pada setiap item)

## 4. Halaman Kontak

### Layout Section-by-Section (Atas ke Bawah):

**A. Header (Sticky)**
- Sama dengan header beranda

**B. Hero Section**
- Banner dengan tema kontak
- Judul: "Hubungi Kami"
- Breadcrumb navigation

**C. Informasi Kontak**
- Judul: "Kontak Kami"
- Alamat lengkap dengan icon lokasi
- Nomor telepon dengan icon
- Email dengan icon
- Jam operasional dengan icon jam
- Social media links dengan icon

**D. Lokasi Sekolah (Map)**
- Embedded map (Google Maps)
- Marker lokasi sekolah
- Informasi lokasi di marker

**E. Formulir Kontak**
- Judul: "Kirim Pesan"
- Field: Nama lengkap, email, nomor telepon, subjek, pesan
- CTA: Tombol "Kirim Pesan"

**F. Tim Kontak Darurat**
- Judul: "Tim Kontak Kami"
- Card untuk setiap tim (Kurikulum, Kesiswaan, Humas, dll.)
- Nama, jabatan, email, dan nomor telepon

**G. Pertanyaan Umum (FAQ)**
- Judul: "Pertanyaan yang Sering Diajukan"
- Accordion dengan pertanyaan dan jawaban terkait kontak

**H. Footer**
- Sama dengan footer beranda

### Urutan Konten:
1. Header
2. Hero Section
3. Informasi Kontak
4. Lokasi Sekolah (Map)
5. Formulir Kontak
6. Tim Kontak
7. FAQ
8. Footer

### Posisi CTA:
1. Formulir Kontak: "Kirim Pesan"
2. Setiap nomor telepon: Link "Telepon"
3. Setiap email: Link "Email"
4. Social media: Link ke akun masing-masing
5. Map: "Dapatkan Arah"
